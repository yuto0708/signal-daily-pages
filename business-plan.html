1. 目的

本計画の目的

本計画の目的は、Threads特化の自動運用システムを販売し、 2026年春ローンチで売上200万円を作ることです。 ただし単に商品を販売するのではなく、今後の高単価化と月額化を見据えた 市場テスト、販売実績獲得、導入者の声の収集も同時に達成目標とします。

今回の位置づけ

今回は最終形の完成版販売ではなく、将来の 機材価格20万円 + 月額課金へ接続するための 先行導入フェーズとして設計する。

2. 商品定義

販売商品の定義

今回販売する商品は、単なる投稿作成ツールではない。Threads運用における 認知、教育、導線の流れを回すための自動運用基盤として位置づける。

現状機能

  • AI投稿生成
  • 予約投稿 / 自動投稿
  • 運用モード切替（full / semi / generate_only）
  • 週間プラン生成
  • 改善戦略生成
  • マルチアカウント対応
  • Threads連携実装済み

商品の本質

  • 投稿数を増やすための道具ではない
  • 10表示レベルのゴミ投稿量産を終わらせるための仕組みである
  • 運用判断の迷いを減らし、導線まで含めて回す基盤である
  • 伸ばした人の知識を分析し、型に落としたこと自体が価値である

3. 市場認識

現状認識と競争環境

市場では、AI自動化、教育ポスト、Threads収益化の訴求が埋まり始めている。 競合は「自動化できる」「投稿が作れる」に寄りがちであり、 ここに真正面から乗ると埋没しやすい。

  • 競合観測: AI自動化、教育ポスト、Threads収益化の文脈が強い。
  • 差別化方針: Threads特化、売れる流れの自動化、ゴミ投稿量産の終了、運用判断の固定化。
  • 参考にする売り方: 実況、証拠、不安解消、思想、限定、販売の流れ。

4. ターゲット

狙う顧客と切る顧客

本命ターゲット

  • 商品はあるがSNS導線が弱い人

  • バズだけ狙いたい人
  • 商品がない初心者
  • 完全放置で全部任せたい人

5. 訴求戦略

主張の芯と販売メッセージ

  • 主張1: 自動化しても伸びない人が多いのは、自動化する前の設計と知識が死んでいるから。
  • 主張2: Threadsは投稿を頑張る場所ではなく、売れる流れを作る場所。
  • 主張3: 勝つのは投稿を頑張る人ではなく、運用を仕組みにした人。
  • 主張4: 知識がない人ほど、カーソル、Claude、周辺ツールに無駄なお金をかけて遠回りする。
販売メッセージの中心

「伸ばした人の知識を分析して、Threads運用の型にした。 無駄なツール課金を増やさず、売れる流れを回せるようにした。 だから、知識がない人でも迷わず運用できる。」

6. 販売導線

ローンチ導線設計

今回はオープンチャットローンチではなく、シンプルな情報導線で販売する。

  • 集客: X・Threads
  • 教育入口: 無料動画配布
  • リスト化: LINE・メルマガ登録
  • 教育: 問題提起、常識破壊、失敗原因の再定義
  • 販売: VSL
  • 決断促進: 限定、期限、人数制限

無料動画の役割

無料動画は教育・興味付け・問題提起・常識破壊の役割を持つ。 「何の商品か」「なぜ必要か」を理解させることが目的。

VSLの役割

VSLは商品説明・差別化・価格正当化・限定訴求・販売の役割を持つ。 無料動画で理解させ、VSLで欲しいに変える。

7. 価格設計

価格・オファー戦略

今回の価格帯は、59,800円〜69,800円で設計する。 本命は通常69,800円、今回のみ先行59,800円の二段階提示である。

項目 内容 狙い
通常価格 69,800円 本来価値のアンカーを作る
先行価格 59,800円 今買う理由を作る
将来価格構想 機材価格20万円 + 月額 今回の先行導入価値を強調する
限定条件 先着・人数制限・初回導入枠 決断を強制的に揃える

8. 売上計画

200万円達成シナリオ

基準ライン

59,800円 × 34本 = 約203万円

69,800円 × 29本 = 約202万円

本命シナリオ

先行59,800円で20本

通常69,800円で10〜13本

合計約189万円〜210万円

考え方

59,800円だけで押し切るのではなく、先行価格で初速を作り、 通常価格で利益を積む構造にする。

現実的な販売目線

500リストがホットではなくても、既存リスト販売 + X/Threads新規流入 + 動画教育 + 締切追い込みで 30本前後までは十分に射程に入る。

9. 販売時期

いつ販売するか

1回目販売

4月30日頃開始、5月2日〜3日締切

GWに間に合わせることで、「休み中に導入したい」「SNS導線を立て直したい」 という欲求を利用する。

2回目販売

5月27日〜31日付近

給料日後に寄せることで決済ハードルを下げ、 かつ月末締切で決断を強く作る。

10. 実行戦略

一般的なローンチに追加する戦略

  • 先行案内希望者の分離取得: 販売前から温度の高い層を別で集める。
  • 人数制限の導入: 期限だけでなく席を理由にする。
  • 実況コンテンツ: 開発実況、改善実況、運用実況を見せる。
  • 診断導線: 「あなたがSNSで疲弊する理由はこれ」という無料診断で自分ごと化を作る。
  • 比較表: 一般的なAI投稿ツールとの違いを明文化する。

11. リスク

主要リスクと対策

リスク 内容 対策
投稿品質の核が弱い Threadsで配られやすい投稿設計への特化がまだ甘い。 万能AIとして売らず、運用基盤として見せる。
ホットでないリスト 500リストあるが、温度が高くない。 無料動画とVSLで教育を厚くし、X/Threads流入も追加する。
高単価耐性 文章だけで59,800円以上は重い可能性がある。 無料動画で理解、VSLで欲求と価格正当化を行う。
期待値過剰 万能感を出すと不一致が生まれる。 向いていない人を先に切り、先行導入の文脈で売る。

12. 結論

最終方針

今回のローンチは、Threads自動化ツールの販売ではなく、 Threadsで売れる流れを仕組みに変える自動運用基盤 の先行導入販売として行う。

  • 販売時期: 1回目はGW、2回目は5月末の給料日後。
  • 販売価格: 通常69,800円、今回のみ先行59,800円。
  • 販売導線: X・Threads → 無料動画 → LINE・メルマガ → VSL → 販売。
  • 訴求軸: 知識の型化、無駄課金の削減、ゴミ投稿量産の終了、売れる流れの構築。
  • 長期方針: 将来的に20万円 + 月額モデルへ接続する。
